# docker-php

Docker && Ubuntu 16.04 && Php && Sqlsrv && Mysqli

## Requirements

* [Docker](https://www.docker.com/)
* [Docker Compose](https://docs.docker.com/compose/)

## About this docker's settings

- ubuntu:16.04
- php 7.0
- sqlsrv
- mysqli

## Installation

```bash
docker-compose up -d --build --force-recreate
```
